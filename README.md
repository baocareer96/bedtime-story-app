# Bedtime Story App

A simple iPad-friendly bedtime story web app built with Next.js, TypeScript, Tailwind CSS, framer-motion, and local JSON story data.

## Quick Start

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Included Features

- Home page with local story cards
- Story reader with one scene per page
- Swipe, keyboard, and button navigation
- Audio button with SpeechSynthesis fallback
- PWA manifest and home-screen-friendly metadata
- Simple story management through `data/stories.json`
